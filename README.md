# Walmart Sales Data Analysis using SQL

## 📌 Project Overview

This project analyzes Walmart sales data using MySQL to uncover valuable business insights. The analysis focuses on product performance, customer purchasing behavior, branch performance, and overall sales trends to support data-driven business decisions.

---

## 📂 Dataset

- **Source:** Kaggle – Walmart Sales Dataset
- **Records:** 1000
- **Columns:** 17
